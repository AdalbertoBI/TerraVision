🚀 COMO COMEÇAR
================================================================================

1. ABRIR APLICAÇÃO
   → Abra: index-new.html
   → No navegador (Chrome, Firefox, Safari, Edge)

2. PRIMEIRA VEZ
   • Permitir acesso à câmera
   • Calibrar o rastreamento (9 pontos)
   • Explorar painel de câmera

3. USAR CÂMERA AVANÇADA
   • Zoom: Slider de 1x até 10x
   • Brilho: Ajuste 0-200%
   • Contraste: Ajuste 0-200%
   • Resolução: Escolha entre 4 opções
   • Screenshot: Clique para baixar PNG

4. USAR FULLSCREEN
   Desktop:
   • Pressione F (atalho)
   • Ou clique botão "📺 Tela Cheia"
   • ESC para sair

   Mobile:
   • Duplo toque na pizza
   • Duplo toque novamente para sair
